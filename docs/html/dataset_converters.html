Converters#

We made available the ShodanDatasetManager and CensysDatasetManager, both class to convert multiples Shodan (JSON) or Censys (Avro) files into a single Delta format, to enrich and manage others aspects of the final dataset. The API Reference page describes all available methods. To create the dataset, run the method convert_files as shown below:

1from tlhop.converters import ShodanDatasetManager
2shodan_mgr = ShodanDatasetManager(org_refinement=True, fix_brazilian_cities=True)
3shodan_mgr.convert_files(INPUT_FILES, output_folder=DEST_FOLDER)
4
5from tlhop.converters import CensysDatasetManager
6censys_mgr = CensysDatasetManager(filter_by_country="Brazil")
7censys_mgr.convert_files(INPUT_SNAPSHOT_FOLDER, output_folder=DEST_FOLDER)

After that, the dataset can be read as any other Spark supported format.

1df = spark.read.format("delta").load(DEST_FOLDER)

Dataset optimizations#

Spark may generate small files over time. Because of that, we also expose a method (optimize_delta) to optimize the dataset by merging small files into a bigger size.

1 shodan_mgr.optimize_delta(DEST_FOLDER)

Delta format supports time travel. In order to support this feature, older files version are kept inside dataset folder. When we ensure that older dataset versions are not needed anymore, we can use the remove_old_delta_versions method to force a removal of these old versions.

1 shodan_mgr.remove_old_delta_versions(DEST_FOLDER)

Obs: The complete demonstration is available at conversion_api_shodan.ipynb and conversion_api_censys.ipynb notebooks.
